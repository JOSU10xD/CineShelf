# Setup Guide

## Prerequisites
- Node.js (v18+)
- NPM or Yarn

## Server Setup
1. Navigate to `server/` directory.
2. Install dependencies: `npm install`
3. Create `.env` file:
   ```
   PORT=8080
   TMDB_API_KEY=your_key_here
   ```
4. Start server: `npm run dev`

## App Setup
1. Navigate to root directory.
2. Install dependencies: `npm install`
3. Run Expo: `npx expo start`
